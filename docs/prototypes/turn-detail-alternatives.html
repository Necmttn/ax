UX thesis
Treat the turn as one continuous file. Blocks are DOM-like ranges in the same document, not separate cards. Hover dims unrelated ranges; references remain clickable.

Inspector

Hover a document range or reference to inspect it.

Navigation Model

First click selects and explains. Explicit open link navigates to skill/file/detail. This avoids losing the session while exploring.

One-Turn Composition Map

Useful takeaway
This answers “what is inside this turn?” without splitting the turn into separate cards. For a huge message, the map gives proportions: context, objective, process constraints, budget, refs, and unresolved atoms.
Interaction
Hover a band to highlight that range in the file viewer. Click locks the range and filters references to that block family.

Block Legend

Reference Lens

Click a reference to see its local relationships.

Why not a full graph?

The graph is most useful as an inspector for selected refs, not as the primary UI. Keep the text primary; show one-hop relationships on demand.
