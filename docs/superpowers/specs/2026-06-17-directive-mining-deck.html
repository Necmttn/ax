04why it keeps improving

Two moves make it self-tuning

Lift, not frequency

P(outcome | ngram) / P(outcome)

Rank n-grams by how much more they lead to a memory write / accepted proposal - not raw count. Markers are learned per user, re-fit each ingest. "Different users, different words" dissolves: the table is yours.

Recurrence = escalation

said once    → memory note
keeps coming back → escalate to hook
stops after   → retire

"It keeps coming back" is the feature: if you restate a rule, the note didn't land → promote it to an enforced gate. The system grades its own directives.
